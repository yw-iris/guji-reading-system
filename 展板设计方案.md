# 展板设计方案

---

## 方案概述

以下 5 套展板用于竞赛展示或图书馆展陈，统一采用新中式风格，以米色、赭石、墨绿为主色调，融合古籍元素与现代信息图表设计。

### 正文要点

**系统架构图描述**：
- 顶层：学生端（Web/平板），展示分级阅读界面
- 中间层：AI 智能引擎，包含 NLP 分词、难度分级、异体字识别、课标匹配四大模块
- 底层：CADAL 古籍数字资源库，接入百万级古籍数字化内容
- 右侧：教师管理端，支持任务布置、进度监控、数据分析

**核心功能**：
1. **AI 三层分级阅读**：根据字词难度、句式复杂度、文化背景知识三个维度自动评估古籍文本难度，匹配到 L1（启蒙）至 L6（进阶）六个等级
2. **异体字智能识别**：自动标注古籍中的异体字并显示对应现代规范字，点击可查看字形演变
3. **课标精准匹配**：将古籍篇章与《义务教育语文课程标准》要求自动关联，辅助教师备课
4. **趣味激励机制**：阅读打卡、积分排行、成就徽章，提升学生持续阅读动力

### 正文要点

**AI 三层分级算法**：
- **第一层 — 字词难度**：基于小学各年级识字量标准，统计生字比例，参考《义务教育常用字表》
- **第二层 — 句式复杂度**：分析句长、嵌套层级、文言虚词密度，计算句式复杂度指数
- **第三层 — 文化背景知识**：评估文本涉及的历史典故、名物制度等背景知识的认知难度
- **综合评分**：加权计算后映射到 L1–L6 六个等级，确保每篇古籍都有精准的推荐年级

**课标匹配算法**：
- 基于 NLP 语义分析技术，提取古籍篇章的核心知识点
- 与《义务教育语文课程标准》学段目标库进行向量匹配
- 自动生成"本篇对应课标知识点"标签，辅助教师将古籍阅读融入常规教学

**异体字识别**：
- 基于深度学习 OCR 模型，训练覆盖 5000+ 常见异体字
- 实时标注古籍页面中的异体字，悬浮显示对应现代规范字
- 支持点击查看字形演变轨迹（甲骨文→金文→篆书→隶书→楷书）

## 展板制作通用建议

| 项目 | 建议 |
|------|------|
| **材质** | 推荐使用 KT 板（轻便）或 PVC 板（耐用），表面覆亚光膜 |
| **字体** | 标题使用毛笔楷体/行楷；正文使用宋体/楷体，字号不小于 24pt |
| **装饰** | 统一使用古籍元素：回纹边框、印章、竹简纹理、水墨晕染 |
| **交互** | 建议在展板右下角放置二维码，链接到系统体验页面或视频介绍 |
| **灯光** | 若在室内展陈，建议在展板上方配置暖色射灯，增强展示效果 |
