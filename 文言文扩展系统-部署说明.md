## 二、文言文扩展系统 · 核心功能

进入任意带「文言文」标签的篇目（如《杨氏之子》《学弈》《桃花源记》），阅读页会自动切换为**专属文言文模式**，包含五大模块：

| 模块 | 说明 |
|------|------|
| **逐句对照翻译** | 原文与白话译文逐句并排，点击文字逐句递进展开，降低阅读门槛 |
| **语法标注** | 句中字词按**实词 / 虚词 / 通假字 / 古今异义 / 词类活用 / 特殊句式**六类着色标注，悬停看释义 |
| **背景拓展** | 出处、作者生平、历史背景、相关典故卡片，并附**思维导图**（MindMap）梳理文章脉络 |
| **专项练习** | 翻译、断句、虚词选择、实词解释、默写填空、文意理解等题型，可交互作答并即时判分 |
| **CADAL 溯源** | 关联 CADAL 古籍善本，展示文献出处与版本溯源 |

> 版本切换卡片（⭐简单版 / ⭐⭐适中版 / ⭐⭐⭐挑战版）沿用原有三档适配逻辑，文言文模式同样支持。

## 四、如何使用

1. 打开首页 → 点击绿色按钮「🎒 我是小学生，开始读古诗！」
2. 在检索页搜索「文言文」或具体篇名（如「学弈」），利用自动联想快速定位
3. 进入阅读页后，系统自动识别「文言文」标签并进入专属模式
4. 从「⭐简单版」起步，逐句点击展开；底部 Tab 切换**字词注释汇总 / 背景拓展 / 文言文练习 / CADAL 溯源**
5. 完成阅读与练习后领取星星奖励，可导出 PDF 或收纳进研学作品
