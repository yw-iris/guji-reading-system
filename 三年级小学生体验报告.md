# 🧒 三年级小学生视角体验报告

> 体验人：假设为小学三年级学生（9岁，识字量约2500字，刚开始接触浅显古文）
> 体验日期：2024年
> 体验方式：从首页进入 → 点击"进入系统" → 选择学生端 → 探索古籍 → 阅读

---

## 一、整体第一印象

### 🟢 好的地方
- **颜色很好看**：棕色、米色像古书的感觉，不刺眼
- **Logo 很酷**：那个红色的"古籍探宝"印章像电视里看到的古代印章
- **数字很大**：2.8M+、1-12、3层这些数字很醒目，虽然不太懂什么意思
- **按钮很大**：首页三个按钮都很大，手指头点起来很方便

### 🔴 不好的地方
- **标题太长了**："义务教育课标智能适配与轻量化重构阅读系统"——我读了三遍也没读完，每个字都认识但连起来不知道什么意思
- **有很多不认识的词**：什么"语义重构"、"轻量化"、"学术检索"、"策展"——语文课上从来没学过这些词
- **不知道从哪开始**：首页有三个按钮，我该按哪个？"进入系统"是什么？

---

## 二、逐页体验

### 第1步：首页（LandingPage）

**我看到的东西**：
- 黑色背景上一个超长的标题（17个字！比我们课文标题长多了）
- 三个按钮：红色"5分钟系统演示"、金色"课标匹配引擎"、浅色"进入系统"
- 四个数字卡片：2.8M+、1-12、3层、5min

**我会怎么想**：
> "这个网站是干什么的？老师让我来读古诗，我该点哪里？"
> "5分钟系统演示是什么？要不要点？"
> "课标匹配引擎又是什么？"
> "算了，点'进入系统'试试吧..."

**问题**：
1. ❌ 标题对三年级学生完全不友好——看不到"古诗"、"读诗"、"学古诗"这些词
2. ❌ 三个按钮的作用不清楚——哪个是给学生的？
3. ❌ "5分钟系统演示"对学生来说没意义

### 第2步：点"进入系统"后

**我看到的东西**：
- 左边多了侧边栏，深棕色
- 菜单：首页、古籍探索、我的阅读、研学任务
- 顶部显示"🎒 小探宝 · 5年级"
- 页面跳到了"古籍探索"

**我会怎么想**：
> "哦，这里有'古籍探索'，应该能看古诗了！"
> "研学任务是什么？研学...我们学校没有研学这个词。"

**问题**：
1. ⚠️ "研学"这个词小学生不懂——应该说"作业"或"任务"
2. ⚠️ 默认角色是"学生"很好，但名字显示"小探宝"有点奇怪
3. ✅ 侧边栏菜单比较简单，四个选项还好

### 第3步：古籍探索页

**我看到的东西**：
- 标题"古籍探索"
- 一个搜索框，里面写着"搜索古籍名称、作者、标签..."
- 年级选择框、朝代选择框
- 12张卡片：静夜思、登鹳雀楼、守株待兔、杨氏之子...
- 每张卡片有朝代标签、年级标签、CADAL编号

**我会怎么想**：
> "太好了！看到静夜思了！这是我们上学期学的！"
> "点了看看..."

**问题**：
1. ✅ 搜索框比较好用，输入"静"就能找到
2. ⚠️ "CADAL: 07100190" 这个编号完全不懂是什么意思
3. ⚠️ 卡片上的标签太多了（朝代+多个年级+课本对应），有点眼花
4. ✅ 卡片可以点，颜色区分了不同朝代，不错

### 第4步：进入阅读页（以静夜思为例）

**我看到的东西**：
- 顶部：返回按钮 + "静夜思" + "唐"标签 + "李白"
- 三个切换按钮：📜 原版 / 📖 简化 / 💡 白话
- 一个图片按钮
- 字体大小：小/中/大
- 课本对应：1年级下学期 · 静夜思
- 阅读区：一句话一句话显示，点击出现下一句

**我会怎么想**：
> "哇，一句一句出现好好玩！像在玩游戏！"
> "按一下出来一句，像在解密！"

**这是整个网站我最喜欢的地方！** 🎉

**但是...**
> "原版是什么？简化是什么？白话是什么？"
> "我点了📜原版，字好难啊，有些字不认识！"
> "换成💡白话，哦！看懂了！"

**问题**：
1. ✅ 逐句阅读模式非常棒！很适合小学生
2. ❌ "原版/简化/白话"三个选项，小学生不知道区别
3. ❌ 切换按钮用 emoji（📜📖💡），对三年级来说不够直观
4. ⚠️ 阅读完成后出现"考点提炼"，"考点"这个词小学生不熟悉
5. ❌ 生字注释在右边，屏幕小的时候看不到

### 第5步：我的阅读页

**我看到的东西**：
- 四个统计卡片：已读古籍、阅读时长、练习题、正确率
- 最近阅读列表
- 我的研学任务
- 猜你喜欢（推荐）
- 积分和兑换

**我会怎么想**：
> "我已经读了1篇了！"
> "积分是什么？可以换东西吗？"
> "兑换...什么意思？"

**问题**：
1. ⚠️ "积分兑换"概念对三年级偏复杂
2. ⚠️ 页面信息量太大——统计+阅读记录+任务+推荐+积分，太长了
3. ✅ "猜你喜欢"推荐区不错

## 三、三年级小学生最严重的 10 个问题

| 排名 | 问题 | 严重程度 | 具体表现 |
|------|------|---------|---------|
| 1 | **首页标题看不懂** | 🔴 致命 | 17字专业术语标题，小学生完全无法理解这个网站是干什么的 |
| 2 | **没有"我是学生"的明确入口** | 🔴 致命 | 三个按钮，学生不知道点哪个 |
| 3 | **太多成人化的专业词汇** | 🔴 严重 | "语义重构"、"轻量化"、"学术检索"、"策展"、"课标匹配引擎" |
| 4 | **三层切换按钮不直观** | 🟡 中等 | emoji + 简短标签，小学生不理解 A/B/C 层的区别 |
| 5 | **生字注释位置不友好** | 🟡 中等 | 在小屏设备上藏在右边，小学生看不到 |
| 6 | **"研学"这个词小学生不懂** | 🟡 中等 | 应该说"学习任务"或"作业" |
| 7 | **CADAL编号无意义** | 🟢 轻微 | 小学生不需要看到这些技术编号 |
| 8 | **阅读完成后的"考点"区域** | 🟢 轻微 | "考点提炼"对三年级来说太成熟 |
| 9 | **积分系统偏复杂** | 🟢 轻微 | 兑换概念需要简化 |
| 10 | **没有语音/朗读功能** | 🟡 中等 | 三年级学生更喜欢听读，而不是默读 |

## 四、改进建议（按优先级）

### 🔴 紧急改进（影响小学生是否能使用）

#### 1. 首页增加"我是小学生"大按钮
```
现在：三个按钮（系统演示、课标引擎、进入系统）
建议：增加第四个最显眼的按钮——
  🎒 我是小学生，开始读古诗！
  （放在最上面，最大，最鲜艳的颜色）
```

#### 2. 简化首页标题
```
现在：基于CADAL古籍资源的义务教育课标智能适配与轻量化重构阅读系统
建议：📜 古籍探宝 —— 和小学生一起读古诗！
副标题：从课本出发，发现好玩的古诗世界
```

#### 3. 三层切换改为人话
```
现在：📜 原版 / 📖 简化 / 💡 白话
建议：🔤 带拼音 / 📖 我能读懂 / 📚 挑战原版
或：⭐ 简单版 / ⭐⭐ 适中版 / ⭐⭐⭐ 挑战版
（用星星等级表示难度，小学生一看就懂）
```

### 🟡 重要改进（提升体验）

#### 4. 菜单用小学生语言
```
现在：古籍探索、我的阅读、研学任务
建议：🔍 找古诗、📖 我读过的、✏️ 我的作业
```

#### 5. 生字注释放在句子旁边
```
现在：生字注释在右侧面板
建议：每个生字直接在句子中高亮，点击弹出小气泡显示拼音和意思
（像课本上的注释一样！）
```

#### 6. 阅读完成后简化
```
现在："考点提炼与练习" + 复杂题目
建议：
  🎉 太棒了！你读完了《静夜思》！
  
  你认识这些字吗？（显示生字卡片）
  来试试小挑战！（1-2道简单选择题）
  
  获得 ⭐ 10 颗星星！
```

#### 7. 增加朗读按钮
```
在每个句子上方增加一个 🔊 朗读按钮
点击后播放这句诗的朗读（可以用浏览器自带语音）
```

### 🟢 锦上添花（让体验更好）

#### 8. 古诗卡片增加难度星级
```
每张古诗卡片上显示 ⭐ 难度（1-5星）
让学生知道这篇难不难
```

#### 9. 积分改成星星/徽章
```
现在：积分数字 + 兑换
建议：收集 ⭐ 星星 → 解锁徽章 → 排行榜
（像游戏一样，更有动力！）
```

#### 10. 增加"每日一首"功能
```
首页或探索页顶部：📅 今日推荐古诗
每天推荐一首适合该年级的古诗
```

## 五、总结

### 当前状态
网站功能很完整，技术实现很扎实，但**语言和交互对小学生不够友好**。一个三年级小学生可能会：
- ✅ 能成功找到古诗并开始阅读
- ✅ 喜欢逐句点击的阅读方式
- ❌ 在首页就感到困惑，不知道这个网站是干什么的
- ❌ 被很多专业词汇吓到
- ❌ 不会主动使用生字注释、考点练习等功能

### 核心矛盾
> 首页是给评委/老师看的（专业、学术），但实际使用者是小学生（需要简单、有趣）。

**建议**：首页保持现在的专业版本用于比赛展示，但增加一个**学生快捷入口**——首页顶部增加显眼的"小学生入口"，点击后进入一个专门为小学生设计的简化版首页。
