# 古籍研学活动手册方案

---

## 一、手册封面设计描述

**风格定位**：国风印章风格，古朴典雅又不失童趣。

| 设计元素 | 描述 |
|---------|------|
| **主色调** | 米黄底色（仿古纸质感），深棕色边框 |
| **标题** | "古籍探秘——小小传承人研学手册"，采用毛笔楷体竖排，烫金效果 |
| **装饰图案** | 四角分布传统回纹边框，中心下方压一枚红色篆刻印章"研学" |
| **辅助元素** | 手绘风格的竹简、毛笔、砚台等文房元素点缀 |
| **底部** | 学校名称、班级、姓名填写栏，横线留白 |
| **尺寸** | A4 竖版（210mm × 297mm） |

## 二、研学活动说明

| 项目 | 内容 |
|------|------|
| **活动主题** | 古籍探秘——让 CADAL 古籍走进课堂 |
| **活动目标** | 1. 激发小学生对中华古籍的兴趣与热爱<br>2. 培养初步的古籍阅读能力和文献意识<br>3. 借助 AI 工具降低古籍阅读门槛<br>4. 提升语文素养和传统文化认知 |
| **适用年级** | 小学 3–6 年级 |
| **活动时长** | 5 个活动，每个活动 40–60 分钟，可分 5 周完成 |
| **活动形式** | 课堂集中活动 + 课后自主探索 + 成果展示 |

## 三、研学活动方案

### 活动一：古籍探宝大冒险

| 项目 | 内容 |
|------|------|
| **活动目标** | 认识古籍的基本形态，了解古籍的分类和特点，激发探索兴趣 |
| **所需材料** | CADAL 古籍阅读系统、古籍图片卡（线装书、竹简、卷轴等）、"探宝任务卡"、彩色贴纸 |
| **活动流程** | **第一步：古籍大观园（10 分钟）**<br>展示不同形态的古籍图片，请学生猜测是什么，教师讲解古籍的基本知识（线装书、蝴蝶装、经折装等）。<br><br>**第二步：发放探宝任务卡（5 分钟）**<br>每人一张任务卡，上面有 5 个"古籍探宝任务"（如：找到一本带插图的古籍、找到一本有红色印章的古籍等）。<br><br>**第三步：系统探宝（20 分钟）**<br>学生在 CADAL 古籍阅读系统中自由浏览，完成探宝任务。每完成一个任务贴一枚贴纸。<br><br>**第四步：分享发现（10 分钟）**<br>请几位同学展示自己找到的"宝藏古籍"，说说为什么觉得有趣。<br><br>**第五步：教师总结（5 分钟）**<br>回顾古籍的种类和特点，表扬完成任务的"探宝小达人"。 |
| **评价方式** | 任务卡完成度（贴纸数量）+ 课堂分享表现 |

---

### 活动二：我是小小校对官

| 项目 | 内容 |
|------|------|
| **活动目标** | 了解古籍中的异体字现象，借助 AI 异体字识别工具完成"校对"任务，培养细心和专注力 |
| **所需材料** | CADAL 古籍阅读系统（异体字识别功能）、"校对任务单"（含原文与现代规范字对照表）、红笔 |
| **活动流程** | **第一步：情境导入（5 分钟）**<br>教师讲故事："古时候，一本书要经过很多人抄写，抄着抄着就会出现一些不一样的字，这些字叫异体字。今天你们就是小小校对官，来帮古籍'纠错'！"<br><br>**第二步：认识异体字（10 分钟）**<br>展示 5–8 组常见的异体字与现代规范字对照（如：體→体、羣→群、峯→峰），请学生观察异同。<br><br>**第三步：AI 辅助校对（20 分钟）**<br>学生在系统中打开指定古籍页面，使用 AI 异体字识别功能，在"校对任务单"上圈出异体字，并在旁边写出对应的现代规范字。<br><br>**第四步：校对成果展示（10 分钟）**<br>小组互相核对校对结果，教师公布正确答案，评选"最佳校对官"。 |
| **评价方式** | 校对准确率（找出异体字数量/总异体字数量）+ 书写规范性 |

---

### 活动三：古籍故事我来演

| 项目 | 内容 |
|------|------|
| **活动目标** | 通过阅读古籍中的故事片段，理解内容后进行角色扮演，提升阅读理解与表达能力 |
| **所需材料** | CADAL 古籍阅读系统（含 AI 分级阅读故事）、简易道具（头饰、扇子等）、剧本卡片 |
| **活动流程** | **第一步：选故事（5 分钟）**<br>小组在系统中选择一篇适合本年级的古籍故事（如《世说新语》中的小故事、《三字经》故事等），使用 AI 白话文翻译辅助理解。<br><br>**第二步：读故事（10 分钟）**<br>小组共同阅读故事，讨论人物关系和情节发展，分配角色。<br><br>**第三步：演故事（20 分钟）**<br>各小组依次上台表演，可使用简易道具。鼓励加入创意改编。<br><br>**第四步：评故事（10 分钟）**<br>观众同学用"小星星"投票选出最佳表演组，教师点评故事寓意。 |
| **评价方式** | 表演完整度 + 对故事内容的理解准确度 + 小组合作表现 |

### 活动四：古籍知识大闯关

| 项目 | 内容 |
|------|------|
| **活动目标** | 以竞赛形式巩固古籍相关知识，激发团队合作和竞争意识 |
| **所需材料** | CADAL 古籍阅读系统（积分/排行榜功能）、答题器或举牌、计分板、奖品（书签/印章） |
| **活动流程** | **第一步：组建战队（5 分钟）**<br>全班分为 4–6 个小组，每组起一个与古籍相关的队名（如"竹简战队""墨香小队"）。<br><br>**第二步：闯关规则说明（3 分钟）**<br>共设 4 关：第一关"古籍知识抢答"、第二关"异体字辨认"、第三关"古籍分类"、第四关"AI 寻宝"（使用系统快速找到指定古籍）。<br><br>**第三步：闯关进行时（25 分钟）**<br>教师主持，逐关进行。每关答对得分，累计积分。<br><br>**第四步：颁奖与总结（7 分钟）**<br>公布积分排行榜，颁发"古籍知识小达人"证书和奖品，总结知识点。 |
| **评价方式** | 团队累计积分 + 个人参与度 |

### 活动五：我的第一本古籍

| 项目 | 内容 |
|------|------|
| **活动目标** | 综合运用所学，亲手制作一本仿古籍小册子，体验古籍制作过程，培养动手能力和创造力 |
| **所需材料** | A4 宣纸或做旧纸张、棉线、针、毛笔/软笔、墨汁、印章、CADAL 古籍阅读系统（参考排版） |
| **活动流程** | **第一步：欣赏古籍之美（5 分钟）**<br>在系统中浏览几本排版精美的古籍，观察版式（天头地脚、界栏、鱼尾等），讨论古籍排版的特点。<br><br>**第二步：设计内容（15 分钟）**<br>学生在做旧纸张上抄写一段自己喜欢的古诗文或自创小故事，注意模仿古籍的竖排书写格式。<br><br>**第三步：装订成册（15 分钟）**<br>教师示范线装书的基本装订方法（四眼订法），学生在教师指导下完成装订。<br><br>**第四步：加盖印章与展示（10 分钟）**<br>为自己的"古籍"加盖印章（姓名章、闲章），封面题写书名。小组内传阅欣赏，评选"最美古籍"。 |
| **评价方式** | 装订完整度 + 书写美观度 + 创意表现 |
